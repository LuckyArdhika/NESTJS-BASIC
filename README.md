# NESTJS-BASIC
NestJs Monorepo Boilerplate/Starter

## Tech Stack
PostgreSQL - Database</br>
ExpressJs - Routing</br>
Prisma - ORM</br>
Jest - Testing</br>
Eslint - Linter</br>
Zod - Validator</br>
JWT - Tokenization</br>
Swagger UI -API Documentation</br>

## Feature
Local Auth (signup, signin, forgot-password)</br>
Seeder</br>
Prisma Migration</br>
User MVC (crud)</br>
Roles Guard (member, admin)</br>
CASL Ability</br>
Logger</br>
Template body</br>
Absoulte Import</br>

## Security
Helmet</br>
CSRF token</br>
CORS</br>
Rate limiter</br>

# How to start
--generate db schema: npx prisma migrate ...
--seeding: npm run seed
